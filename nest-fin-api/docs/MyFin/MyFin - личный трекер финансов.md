Отслеживание расходов и доходов с разбивкой по месяцам

У каждого пользователя есть свой баланс

Доходы: Мы можем добавить доход, указать категорию откуда он пришёл, сумма зачисляется на баланс
Расходы: Мы можем добавить запись траты, указать категорию куда были потрачены деньги, сумма списывается с баланса

Вся история разбивается по месяцам
