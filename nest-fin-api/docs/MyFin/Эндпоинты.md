### Транзакции

| Метод  | URL                                       | Что делает                           |
| ------ | ----------------------------------------- | ------------------------------------ |
| GET    | /transactions?type=&from=&to=&categoryId= | Список с фильтрами                   |
| GET    | /transactions/:id                         | Одна транзакция                      |
| POST   | /transactions                             | Создать (+ обновить баланс атомарно) |
| PATCH  | /transactions/:id                         | Обновить всё кроме type и amount     |
| DELETE | /transactions/:id                         | Удалить (+ обновить баланс атомарно) |
| GET    | /transactions/summary?from=&to=           | Суммы за период                      |
| GET    | /transactions/by-category?from=&to=       | Разбивка по категориям               |

### Категории

| Метод  | URL             | Body       | Ответ                           |
| :----- | :-------------- | :--------- | :------------------------------ |
| GET    | /categories     | —          | `Category[]` (мои + глобальные) |
| POST   | /categories     | `{ name }` | `Category` (новая)              |
| PATCH  | /categories/:id | `{ name }` | `Category` (обновлённая)        |
| DELETE | /categories/:id | —          | `204` или `{ success: true }`   |
